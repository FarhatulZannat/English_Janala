Frequently Asked Questions

1. What is the difference between var , let and const?
var function-scope ও re-declare করা যায়, let block-scope এবং re-declare করা যায় না, const block-scope এবং value change করা যায় না।
What is the difference between map() ,forEach() and filter()?
map() নতুন array return করে, forEach() কিছু return করে না শুধু iteration করে, filter() শর্ত মিলে এমন elements দিয়ে নতুন array বানায়।
What are arrow functions, and how are they different from regular function?
Arrow functions short syntax, no own this, and can't be used as constructors, unlike regular functions।
How do javaScript Promise work?
A JavaScript Promise represents an asynchronous operation that can be pending, fulfilled, or rejected, and allows handling the result or error when ready.
How does closure work in JavaScript?
A closure is a function that remembers and can access variables from its outer scope even after the outer function has finished execution.
